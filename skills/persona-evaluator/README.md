# evaluator-skill

> Score any OpenPersona persona pack across 4 Layers + 5 Systemic Concepts.

**evaluator-skill** is the quality audit tool for the [OpenPersona](https://openpersona.co) framework. It produces a 9-dimension report (Soul · Body · Faculty · Skill · Evolution · Economy · Vitality · Social · Rhythm) with scores, constitution compliance check, and actionable improvement recommendations.

## Quick Start

```bash
npx openpersona evaluate <slug>
npx openpersona evaluate <slug> --json
```

## Score Bands

| Score | Band |
|-------|------|
| 9–10 | Excellent |
| 7–8 | Good |
| 5–6 | Developing |
| 0–4 | Needs Work |

## Install

```bash
npx openpersona skill install acnlabs/evaluator-skill
```
