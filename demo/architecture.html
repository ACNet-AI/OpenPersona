④ Lifecycle Protocol — Nervous System
START
openpersona state read <slug>
→ state-runner.js → state-sync.js read
inject: mood / relationship / evolvedTraits
process: pendingCommands queue
MID
Signal Protocol (persona → host)
state-sync.js signal <type> '{...}'
writes signals.json · reads signal-responses.json
types: scheduling · file_io · tool_missing · …
END

append eventLog (max 50) · clear pendingCommands

HOST→

capability_unlock · context_inject
trait_nudge · relationship_update
